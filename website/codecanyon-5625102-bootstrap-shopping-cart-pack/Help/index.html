B - Installation - top

1- Copy all the content on your server (make sure you have php 5.3 + & Mysql 4.0+)
2- Open "config.php" (into /BootstrapShoppingCart)and edit the values ​​that you want. Important configure MySql / Paypal / Currency / Shipping
3- Upload the files on your server. Not recommended upload Bootstrap folders (css/img/js)
4- install the mysql schema into your MySql(mysql.sql)
5- Change the image /images/logo_header.png used in email ticket & thanks for purchase page



C - Install into your PHP page - top

Put in front of the code 

<?php 
require_once "BootstrapShoppingCart/config.php"; // load settings file
require_once "BootstrapShoppingCart/BootstrapShoppingCart.php"; // load class shoppingcart
?>
Loads the styles 
<link rel="stylesheet" href="BootstrapShoppingCart/BootstrapShoppingCart.css">
<link rel="stylesheet" href="BootstrapShoppingCart/animation.css">
Loads the JS 
<script>
$(document).ready(function() {                
	$("#BSCart-Horizontal").load("BootstrapShoppingCart/addproduct_horizontal.php");
});            
</script>
